# POINT
Eine ganzzahlige Koordinatenangabe auf dem Bildschirm.

## Definition

```ack
struct
{
	int x, y;
} POINT;
```

## Elemente
- **`x, y`**
	Die Position in ganzen Pixeln. Der Nullpunkt befindet sich in der linken, oberen Ecke. Die X-Achse erstreckt sich nach rechts, die Y-Achse nach unten.
